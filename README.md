# Java OOP RPG Project

## Description
This project is a simple RPG game developed using Java Object-Oriented Programming concepts.

## Concepts Used
- Abstraction
- Inheritance
- Polymorphism
- Encapsulation
- Arrays of objects

## Classes
- Character (Abstract)
- Warrior
- Mage
- Archer
- Ability
- DemoRPG

## Features
- Character creation
- Abilities system
- Combat simulation
- HP and damage management
